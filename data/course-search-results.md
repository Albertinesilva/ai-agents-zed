Erros e Observações:
- DB2: Nenhum curso específico encontrado na Alura. As buscas retornaram principalmente fóruns de dúvidas e tópicos sobre conexão com DB2, mas não cursos estruturados. Recomenda-se buscar em outras plataformas ou verificar cursos de "Banco de Dados" genéricos que mencionem DB2.
- EJB 3.x: Não foram encontrados cursos específicos de EJB 3.x na Alura. O firecrawl retornou fóruns sobre um curso antigo "EJB: O poder da Java EE" e tópicos sobre Jakarta EE. Para a vaga que menciona EJB 3.x, recomenda-se buscar cursos de "Java EE" ou "Jakarta EE" na Alura, ou considerar que o conhecimento em Spring Boot (que o usuário já possui) pode ser transferível para alguns conceitos de EJB.
- Instrutores: Para a maioria dos cursos de front-end (Angular, React, JavaScript), os instrutores principais são Vinny Neves, Nayanne Batista e Neílton Seguins. Para SQL Server, os instrutores são da Escola de Dados (Guilherme Silveira, David Neves, etc.).
- Firecrawl CLI: Funcionou corretamente para as buscas principais (Angular, JavaScript, SQL Server, React). O fallback para fetch foi usado para refinar os dados.
